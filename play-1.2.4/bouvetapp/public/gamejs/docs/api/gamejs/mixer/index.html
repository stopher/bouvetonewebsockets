Module gamejs/mixer

Playing sounds with the html5 audio tag. Audio files must be preloaded with the usual gamejs.preload() function. Ogg, wav and webm supported.

Sounds & Images are loaded relative to './'. You can change that, by setting $g.resourceBaseHref in your page before loading the GameJs modules:

<script>
var $g = {
   resourceBaseHref: 'http://example.com/resources/'
}
</script>
<script src="./gamejs-wrapped.js"></script>
....

Note: sound doesn't work very well across browsers yet. Firefox is best, but all still have bugs.
