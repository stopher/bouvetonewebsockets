Module gamejs/time

Provides tools for game time managment.

This is very different from how PyGame works. We can not pause the execution of the script in Browser JavaScript, so what we do you do is write a main function which contains the code you would put into your main loop and pass that to fpsCallback():

Example 

function main() {
        // update models
        // draw to screen
     };
     gamejs.time.fpsCallback(main, this, 30);
     ;
     function aiUpdate() {
        // do stuff that needs low update rates
     }
     gamejs.time.fpsCallback(aiUpdate, this, 10);

You are encouraged to use multiple `fpsCallbacks()` for the different
kind of update rates you need, e.g. one at 30 fps for rendering and
another at 10 fps for the AI updates.

Parameters

Object thisObj `this` will be set to that object when executing the function
Number fps specify the framerate by which you want the callback to be called. (e.g. 30 = 30 times per seconds). default: 30
