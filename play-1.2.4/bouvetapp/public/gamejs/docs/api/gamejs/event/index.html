Module gamejs/event

Methods for polling mouse and keyboard.

Call get() in your main loop to get a list of events that happend since you last called.

A pattern for using the event loop : your main game function (tick in this example) is being called by gamejs.time.fpsCallback() 25 times per second. Inside tick we call gamejs.event.get() for a list of events that happened since the last tick and we loop over each event and act on the event properties.

var events = gamejs.event.get()
events.forEach(function(event) {
   if (event.type === gamejs.event.MOUSE_UP) {
     gamejs.log(event.pos, event.button);
   } else if (event.type === gamejs.event.KEY_UP) {
     gamejs.log(event.key);
   }
});

Note that some events, which would trigger a default browser action, are prevented from triggering their default behaviour; i.e. event.preventDefault() is called on those specific events, not on all events.
