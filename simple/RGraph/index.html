Faster websites with RGraph

RGraph is a HTML5 Javascript charts library. Using the new HTML5 canvas tag, RGraph creates Javascript charts in the web browser, meaning quicker pages and less web server load. Using RGraph is an easy way of speeding up your website.

RGraph produces a wide variety of Javascript charts - examples and documentation below.

Supported charts

RGraph supports over twenty different types of Javascript charts, including:

  • Bar charts
  • Bi-polar charts
  • Donut charts
  • Fuel gauges
  • Funnel charts
  • Gantt charts
  • Gauge charts
  • Radar charts
  • Radial Scatter charts
  • Rose charts
  • Scatter charts
  • Thermometer charts
  • Vertical Progress bars
  • Waterfall charts
  • Horizontal Bar charts
  • Horizontal Progress bars
  • LED grids
  • Line charts
  • Meter charts
  • Odometers
  • Pie charts

In addition there are variants available on many charts. There is also a growing range of animations and visual effects.
